# 📊 Interactive Sales Dashboard in Excel

## Overview

After completing my Data Analytics Internship, I continued applying and strengthening the skills I learned through hands-on projects.

This project showcases an Interactive Sales Dashboard built in Microsoft Excel using a real-world sales dataset. The dashboard transforms raw transactional data into meaningful business insights through interactive visualizations, KPI tracking, and dynamic filtering.

The objective was to create a user-friendly dashboard that enables quick analysis of sales performance, customer behavior, product trends, and revenue metrics.

---

## Dashboard Features

### 📌 KPI Cards
- Total Sales
- Gross Income
- Average Sales
- Average Income
- Total Transactions

### 📈 Interactive Visualizations
- Sales by Branch
- Sales by Product Line
- Sales by Customer Type
- Sales by Gender
- Payment Method Distribution
- Gross Income by City

### 🎛️ Interactive Filters (Slicers)
- Branch
- City
- Gender
- Product Line
- Payment
- Customer Type
